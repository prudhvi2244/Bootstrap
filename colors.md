# Bootstrap 4 Colors
# Text Colors

* Bootstrap 4 has some contextual classes that can be used to provide "meaning through colors"
* The classes for text colors are

    * .text-muted
    * .text-success
    * .text-danger
    * .text-primary
    * .text-secondary
    * .text-info
    * .text-white
    * .text-dark
    * .text-light
    * .text-warning


# Background Colors

* The classes for background colors are

    * .bg-primary
    * .bg-success
    * .bg-info
    * .bg-warning
    * .bg-danger
    * .bg-secondary
    * .bg-dark
    * .bg-light
